Quick Start.

If you already know the game of Bridge, here are the minimum instructions necessary to get you started playing ZBridge.


Playing a Practice Hand.

When you start ZBridge the first time, you are in Practice Mode.  You can select “New Session” from the toolbar to deal a new hand, and you will automatically be taken through the bidding and card play stages. See Mainwindow. Then upon completion of card play, your scoring will be shown and a new hand will be dealt.


Playing a Rubber.

If you want to play a rubber match, you must select Options from the menu and then Game to show the Game dialog. In the Game dialog you select the Scoring tab and then "Rubber Bridge". This will start a scored rubber match.  Hands will be dealt and played until one side wins the rubber, at which point the total score will be shown and a new scored rubber match started.
You can check the rubber score at any time by selecting Score from the menu.


Playing Against the Computer.

The score dialog mentioned above in "Playing a Rubber" has two more choices you can select if you want to compare your play with what the computer would do. One is with International Match Point (IMP) scoring and the other is with Match Point (MP) scoring. At the end of the play the scoretable shows your play against how the computer would play the cards.


Playing Against Preplayed Games.

On the internet you can find many preplayed bridge games. They are normally stored in the standard PBN format. With ZBridge you can play games from a PBN file and compare your results with the corresponding results in the PBN file. You open the file using the Open entry in the File menu. The game mode depends on the number of results in the PBN files.

 · Practice
When the PBN file has no results.

 · Team Match
When the PBN file has more than one result per board.

At the end of the play the scoretable shows your play against the alredy played games in the PBN file.


Re-Deal / Re-Bid / Re-Play.

You can always deal a new hand by selecting "Deal New Hand", regardless of which mode you are in.  You can also return to the start of bidding with the current hand by selecting "ReBid".  In addition, selecting "RePlay" will return you to the start of card play (i.e., just before the first trick). These choices can be selected from the menu.


Options.

You can select various options from the Options menu to change bidding and play options.


Save/Load.

You can save the current hand and load saved hands using the File menu.
