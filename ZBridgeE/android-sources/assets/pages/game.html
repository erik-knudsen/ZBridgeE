Game Menu Commands

Deal New Hand

Use this command to deal the next hand in cases where you want to skip the current hand.

Activate Deal Profile

Use this command to activate the deal profile. The computer then deals cards according to the given deal profile. For more information about deal profiles see the topic Deal Profile Dialog.

Rebid

Use this command to restart the current game with the bidding phase. This is only possible when the computer is standalone or server.

Replay

Use this command to to restart the current game with the playing phase. This is only possible when the computer is standalone or server.
