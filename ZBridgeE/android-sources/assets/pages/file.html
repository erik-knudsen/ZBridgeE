File Menu Commands

Open

Use this command to open a previously saved game.

Save

Use this command to save the current game using its current name and directory.  When you save a document for the first time, ZBridge displays the Save As dialog box so you can assign a name to your game. 
If you want to change the name and directory of an existing document before you save it, choose the Save As command.

Save As

Use this command to save and name the current game. ZBridge displays the Save As dialog box so you can assign a name to your game.
When you save a game, it will be saved in the PBN format.
To save a game with its existing name and directory, use the Save command.

Delete

Use this command to delete the currently opened game. Only the part played by means of ZBridge will be deleted. The original contents of the PBN file is not touched.

Exit Menu

Use this command to exit from the main menu.
